# v-if和v-for哪个优先级更高
- v-for的优先级更高
- v-for的每次循环都会调用v-if的判断
- 官网推荐将v-if移到父元素中
  
# 组件中的data为什么要用return返回
- 组件是可以复用的，当多个地方引用同个组件的时候，为了防止数据污染，需要将data用return返回
- 使用return返回数据后，我们得到的是调用data函数后生成的数据副本，不会导致数据污染
